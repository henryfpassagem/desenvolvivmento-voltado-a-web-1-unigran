Explicação das tags

:
  • !DOCTYPE: Define o tipo de documento e a versão do HTML que está sendo usada.
  • html: Define o elemento raiz de uma página HTML.
  • head: Contém metadados sobre o documento, como o título da página, links para folhas de estilo e scripts.
  • meta: Fornece metadados sobre a página, como a codificação de caracteres e a descrição do conteúdo.
  • title: Define o título da página, que aparece na guia do navegador ou na barra de título.
  • body: Contém o conteúdo visível da página, como texto, imagens, links e outros elementos.
  • h1, h2: São cabeçalhos de diferentes níveis, onde h1 é o mais importante.
  • p: Define parágrafos de texto.
  • ul e ol: Criam listas não ordenadas e ordenadas, respectivamente.
  • li: Define itens de lista em listas não ordenadas ou ordenadas.
  • div: É usado para criar divisões ou contêineres para organizar o conteúdo.
  • iframe: é usada para incorporar um documento HTML dentro de outro documento HTML. Ela cria um espaço retangular onde pode-se exibir uma página da web externa ou outro conteúdo incorporado, com vídeos, mapas, etc.

As tags HTML apontadas acima são apenas algumas das mais comuns e que foram utilizadas para implementação dos arquivos aulas1_2.html e aula3.html. Além das tags acima podemos citar outras tags importantes como: link, style, a, img, span, table, tr, td, form, input, textarea, hr, button e várias outras. Cada tag desempenha um papel importante na estrutura e no conteúdo de uma página da web.
